# SOAI 官网原型

## 1. 定位

这是 soai.yun 官网首版静态原型，用于承接：

- SOAI 马术智能训练系统介绍
- AI 训练报告
- 教练复核机制
- 马术充气护甲
- 马术教学耳机
- 俱乐部合作
- 内测申请

## 2. 打开方式

可直接打开：

```text
官网原型soai/index.html
```

也可以本地启动：

```bash
cd 官网原型soai
python3 -m http.server 8790
```

访问：

```text
http://localhost:8790
```

## 3. 视觉方向

参考图：

```text
../执行包/assets/官网首页视觉参考.png
```

设计关键词：

- 专业

- 白色和石墨色为主

- 避免夸张营销和绝对安全承诺

## 4. 当前限制

- 表单为原型交互，不提交真实数据。
- 还未接入后台 CRM 或小程序内测报名。
- 产品图片为生成式视觉参考，不是最终商品摄影。
- 上线前需要补充备案、隐私政策、用户授权和真实联系方式。
