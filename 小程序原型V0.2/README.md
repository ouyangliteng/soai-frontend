# 马术 AI 教学小程序 V0.2 原型

## 1. 原型目标

这个工程用于演示 V0.2 内测可用版的核心链路：

学员资料 -> 视频上传 -> AI 分析中 -> 训练报告 -> 历史趋势 -> 教练复核 -> 内测反馈

当前版本使用本地 mock 数据，不依赖真实后端和真实 AI 服务。后续可按 `../执行包/数据结构与接口草案.md` 替换为真实接口。

## 2. 页面

- `pages/home/home`：训练首页
- `pages/profile/profile`：学员资料
- `pages/upload/upload`：视频上传
- `pages/privacy/privacy`：隐私与视频使用说明
- `pages/analysis/analysis`：AI 分析中
- `pages/report/report`：AI 训练报告
- `pages/feedback/feedback`：学员/教练内测反馈
- `pages/trends/trends`：历史趋势
- `pages/coach/coach`：教练工作台
- `pages/coach-student/coach-student`：教练查看学员详情

## 3. 如何打开

1. 打开微信开发者工具。
2. 选择导入项目。
3. 项目目录选择 `小程序原型V0.2`。
4. AppID 可使用测试号或游客模式。
5. 编译后从“训练”页开始体验。

## 4. 已覆盖能力

- 学员资料字段：姓名、年龄、身高、体重、骑龄、当前等级、所属教练、俱乐部。
- 上传体验：资料校验、视频选择、大小/时长校验、视频用途确认、上传进度模拟。
- 隐私说明：上传页可查看视频使用说明，区分训练分析授权和匿名案例授权。
- 分析体验：任务状态保留、分析中、生成报告、失败重试入口。
- 报告内容：评分、问题点、风险点、本次进步、下次训练重点、教练复核和训练重点确认。
- AI 辅助：报告页可生成学员解读和教练复核草稿。
- 历史趋势：最近 5 次/10 次训练趋势切换。
- 教练端：工作台统计、待复核报告、学员详情、反复问题和风险关注。
- 内测反馈：学员/教练身份切换、评分、问题标签、补充说明和 API 提交。
- 埋点闭环：资料、上传、分析、报告、趋势、复核和报告深度阅读关键事件。

## 6. 当前限制

- 未接入真实视频上传服务。
- 未接入真实 AI 视频分析模型。
- 未接入登录和权限控制。
